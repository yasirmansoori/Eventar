## Setup

### Pre-requisites

- *Node:* `^18.0.0`
- *Yarn*

### Install

Clone the repository and create a local branch:

```sh
git clone 
cd eventar

git checkout -b my-branch
```

Install dependencies:

```sh
npm install 
```

## Developing

```sh
# Create App.tsx file in the src folder and start coding
touch src/App.tsx
```

```sh
# launch the playground
npm run dev

# Check Pretty format
npm format:check

# Format the code
npm format
```
